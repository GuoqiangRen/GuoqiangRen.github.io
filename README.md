# Guoqiang Ren Academic Website — Version B

Version B keeps the Deepak Pathak-style information architecture but refines spacing, typography, publication rows, and responsive behavior. It is intended to look slightly more polished while remaining recognizably academic and minimal.

## Preview locally

Open `index.html` directly, or run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy with GitHub Pages

Upload every file in this folder to the repository root, then enable GitHub Pages from the `main` branch and `/ (root)` folder.
